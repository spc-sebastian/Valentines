A whimsical, interactive web application designed to ask my wonderful wife, Makayla, to be my Valentine.

## Features

- **Mischievous "No" Button**: An elusive, dodging button with over 30 funny quips and "sassy" animations to make the proposal experience playful and memorable.
- **Dynamic Sound Effects**: Custom-synthesized Web Audio API sounds for interaction feedback.
- **Romantic Transition**: A curated, multi-phase message sequence that leads to the final question.
- **Personalized Celebration**: A beautiful finale featuring a romantic illustration and a heartfelt message.
- **Heart-Filled Background**: A responsive, floating heart animation that sets the mood.

## Tech Stack

- **Frontend**: React 19, Tailwind CSS
- **Icons**: Lucide React
- **Animations**: CSS Keyframes & Framer-vibe transitions
- **Audio**: Web Audio API (custom oscillators)

## How to Run Locally

1. Clone this repository.
2. Open `index.html` in your browser (using a local server like Live Server or `npx serve .`).
